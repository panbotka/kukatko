# Remove the "Kukátko" brand link from the navbar

The top navigation shows the "Kukátko" brand text as a link, immediately next to the "Knihovna" (Library) link. The brand link is redundant — remove it. The Library remains reachable via the adjacent Knihovna nav link.

## Where it is
- `web/src/components/Layout.tsx`, around lines 227–230: the `<Navbar.Brand as={Link} to="/" title={t('nav.titles.home')}>{t('app.name')}</Navbar.Brand>` block (`t('app.name')` renders "Kukátko"). The Library link is the first entry of `PRIMARY_ITEMS`, rendered right after it in the `<Nav className="me-auto">`.

## Requirements
- Remove the `Navbar.Brand` element so the brand text/link no longer appears in the navbar. The nav links (Library first) should sit where they belong; make sure the navbar layout/spacing still looks right after removal (the `Container`/`Navbar.Collapse`/toggle structure must remain valid and the mobile toggle must still work).

- Clean up cleanly: if `t('nav.titles.home')` or other strings become unused only because of this removal, remove the now-dead references (but do not remove `app.name` if it is used elsewhere — check first, e.g. document title / other components).
