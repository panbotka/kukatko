# Photo detail: organize panel beside description & place (25/75)

On the photo detail page the "Uspořádání" panel and the "Popis a místo" panel are stacked. Put them side by side instead.

## Requirements

- On wide viewports the two panels sit in one row: **"Uspořádání" 25 % width, "Popis a místo" 75 %**.
- Below the Bootstrap `lg` breakpoint they stack full width, in the current order — the detail page must stay usable on a phone.
- Both panels align to the same top edge; the row does not force equal heights that leave a large empty box under the shorter panel.
- No horizontal scrollbar at any viewport width.

## Implementation Notes

- `web/src/pages/PhotoDetailPage.tsx` composes `OrganizePanel` and `MetadataPanel`; use the react-bootstrap grid (`Col lg={3}` / `Col lg={9}`) rather than custom CSS widths.
- Verify visually, not just by tests — a static HTML harness with the real Bootswatch Superhero theme + `app.css` driven through `agent-browser` is the established way in this repo (no auth, no server restart needed).
